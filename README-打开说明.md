# 西樊各庄研学路线 HTML 交互展示版

打开方式：直接双击 `index.html`。

已支持的离线交互：
- 12 个月份研学主题切换。
- 时令推荐路线一键应用到地图选点。
- 地图点位点击弹出详情卡片。
- 详情卡片中加入/移除研学路线。
- 右侧路线列表同步更新。
- 生成研学环线展示提示与线路高亮。
- 当季热门研学路线卡片可同步到地图选点。

说明：
- 这是从当前网页整理出的 HTML 交互展示版，适合现场快速展示。
- 已包含所需图片、视频、地图、点位图标、轮播图片、CSS 与 `static-interactions.js`。
- 线路为展示级人工连线，不具备真实导航精度。
- 如需继续深度开发，请使用原 React 源码工程。

目录结构：
- `index.html`：展示页入口。
- `static-interactions.js`：离线交互脚本。
- `assets/`：网页图片、视频、点位图标、轮播图。
- `app/`：页面样式文件。
- `源码与素材清单.md`：当前交付内容说明。
